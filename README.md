# Portfolio Website

This is a simple static portfolio website.

## Structure

- `index.html` – single-page website including Home/introduction, About Me, Portfolio, Publications, and Contact sections.
- `styles.css` – basic styling and layout adjustments (fixed header, smooth scrolling).

## Getting Started

1. Open `index.html` in your browser to view the site.
2. Replace placeholder text and links with your actual information.
3. Add more sections or projects as needed.

## Deploying

You can deploy this on GitHub Pages by:

1. Pushing the repository to GitHub.
2. Enabling GitHub Pages in the repository settings, serving from the `main` branch.

Enjoy building your portfolio!
